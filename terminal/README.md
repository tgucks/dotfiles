# Terminal Themes

Dracula color profiles for iTerm2 and Terminal.app.

## iTerm2

1. Open Preferences (`Cmd + ,`)
2. Go to Profiles > Colors
3. Click Color Presets > Import
4. Select `Dracula.itermcolors`
5. Choose "Dracula" from the Color Presets dropdown

## Terminal.app

1. Double-click `Dracula.terminal` to import
2. Open Preferences (`Cmd + ,`)
3. Go to Profiles
4. Select "Dracula" and click "Default"
